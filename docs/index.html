Winnow

A self-hosted AI assistant that triages your Fastmail inbox — files the noise, surfaces what matters. Separating the grain from the chaff.

→ Read the full user guide

What it does

  • Connects to Fastmail over JMAP (no fragile IMAP).
  • Cheap-first classification: free header heuristics catch the obvious bulk; only ambiguous mail reaches Claude.
  • Sorts into editable categories → folders; flags important mail; never deletes.
  • Safety bias: low-confidence mail stays in the inbox, so nothing important is hidden.
  • Graduates recurring senders into your own Fastmail Sieve rules (with approval) — server-side, free, always-on.
  • Assisted unsubscribe (opt-in) and a once-a-day digest.
  • A clean, mobile-friendly dashboard over Tailscale or a Cloudflare Tunnel.

Quickstart

docker run --rm ghcr.io/virajchitnis/winnow hashpw   # generate APP_PASSWORD_HASH
cp .env.example .env                          # fill in tokens
docker compose up -d                          # starts in safe dry-run mode
First run seeds the preset categories and previews classifications without moving anything. Turn off dry-run in Settings when you're happy.

Privacy

Your Fastmail token and mail storage stay on your server. For ambiguous mail only, Winnow sends the subject, sender, and a short snippet (never full bodies) to Anthropic to classify — and you can tighten that to subject+sender-only.
